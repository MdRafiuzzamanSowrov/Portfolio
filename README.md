# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MdRafiuzzamanSowrov/Portfolio.git
```

Go to the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

# 🧪 Build for Production

```bash
npm run build
```
